Essa é uma IA para reconhecer documentos como RG, CNH e CPF (futuramente será implementado mais tipos). Lembre-se de fazer o download do Tesseract e mudar o caminho do tesseract nessa parte do arquivo lerImagem.py pytesseract.pytesseract.tesseract_cmd = r'C:\Program Files\Tesseract-OCR\tesseract.exe'. 
Está sendo usando as bibliotecas, Sckit-Learn, Tesseract, cv2 e joblib
Espero que esse codigo possa ajudar de luz para algumas pessoas que estão começando na area de IA/ML.
Estou disponivel para qualquer duvida e sugestão também.
Documentação do Tesseract para download (https://github.com/tesseract-ocr/tessdoc)
